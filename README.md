# Customer Purchase Prediction

## 📌 Project Overview
This project predicts whether a customer is likely to purchase a product based on their data using Machine Learning.

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Random Forest

## 📂 Project Structure
- train.py → Model training
- predict.py → Prediction script
- preprocessing.py → Data cleaning

## 🚀 How to Run
1. Clone the repository
2. Install requirements
3. Run train.py
4. Run predict.py

## 📊 Model Performance
Accuracy and classification report are shown after training.
